# 课程管理系统 java93

## 项目概述

课程管理系统是一款基于B/S架构的web应用程序，采用MVC设计模式，实现了前后端分离的开发模式。系统致力于提供一站式的课程管理解决方案，支持管理员、学生、教师等多角色操作，具备完善的功能模块。

## 技术栈

- **后端：** Java、SpringBoot
- **前端：** Vue
- **数据库：** Mysql
- **开发工具：** IDEA、JDK1.8、Maven

## 系统角色

系统包含以下三种角色：

- 管理员
- 教师
- 学生

## 功能模块

### 管理员模块

- 首页
- 系管理
- 专业管理
- 办理管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

### 教师模块

- 首页
- 授课查询
- 教师课表
- 成绩查询

### 学生模块

- 首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护
